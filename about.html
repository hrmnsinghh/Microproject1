Functionality Overview

When the user clicks the "Load Data" button, the JavaScript code sends a GET request to the server's endpoint (`/api/details`). The server responds with JSON data, which is then dynamically displayed on the page in a well-formatted table.

Implementation Details

  • Server Side (Express): The server is set up using Express.js to serve static files and handle API requests. The JSON data is read from `data/data.json` and served at the `/api/details` endpoint.
  • Client Side (HTML/JavaScript): The client-side code includes an HTML button and JavaScript logic to fetch data from the server and display it in a table. The JavaScript uses `fetch` with async/await to handle the asynchronous request.
